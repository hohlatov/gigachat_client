## 🛠 Стек технологий

- **React** 19.2.4
- **TypeScript** 4.9.5
- **React Router DOM** 7.14.0
- **State Management**: Context API + useReducer
- **Styling**: CSS Modules
- **Markdown**: react-markdown 10.1.0
- **Code Highlighting**: rehype-highlight 7.0.2
- **HTTP Client**: Fetch API
- **Build Tool**: Create React App 5.0.1
